# Chrome Web Store Listing Assets

이 폴더는 스토어 리스팅 업로드용 이미지를 둡니다. Git에 커밋해 팀원이 공유.

## 필수

- `screenshots/01-drives.png` 1280x800 — 드라이브 목록 화면
- `screenshots/02-filelist.png` 1280x800 — 파일 리스트 (정렬/선택)
- `screenshots/03-context-menu.png` 1280x800 — 우클릭 메뉴
- `screenshots/04-progress.png` 1280x800 — 진행률 토스트
- `screenshots/05-conflict.png` 1280x800 — 충돌 다이얼로그

## 선택

- `promo-small.png` 440x280 — Small promo tile
- `promo-marquee.png` 1400x560 — Marquee
- `promo-large.png` 920x680 — Large promo

## 촬영 순서 (수동)

1. Chrome에서 확장 로드 + 새 탭
2. F12로 DevTools → Device Toolbar → 1280x800 고정
3. 각 화면 구성 후 PrintScreen 또는 snipping tool
